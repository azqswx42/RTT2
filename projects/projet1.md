## Application web de gestion de cinéma

Projet de type **application web** permettant de gérer :

- les films
- les séances
- les salles

### Objectifs techniques

- Structurer un backend propre
- Gérer les données de manière fiable
- Proposer une interface claire pour l’utilisateur.
